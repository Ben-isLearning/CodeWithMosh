Font Squirrel

Download a font for free!
The download will most likely be in a .ttf format - Which is fine
But the filetype is far too big to use on a website
Convert that sucker! Using the "Generator" tab of Font squirrel
Now we have lovely .woff and .woff2 text files.
Now Extract and "install" (Move the files) into a new folder within the font directory.
Now install the .CSS file that came with the files
Now copy the @font-face declarations into the very TOP of our projects CSS file
The font-family: "xxx"; declaration names the font family; It can be renamed.
Ammend the SRC urls inside the CSS
Ammend the font-weight on the bold fonts + the font-style on the italic fonts.

Now the new font has been installed, we're ready to use it as normal

++ Flash of Unstyled font
When users have trash connection ~ it'll take a hot second or two before the styling loads
to overcome anything going sideways while this occurs
Add the property; font-display:fallback; to the @font-face declarations.
(or font-display:optional;) If the custom-font does not download in a timely manner - it is forgotten about.
