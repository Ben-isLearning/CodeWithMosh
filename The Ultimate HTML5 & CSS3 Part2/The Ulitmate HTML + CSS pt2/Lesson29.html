Art Direction

Can serve different cropped / zoomed version of image dependant on screen size

Use <picture> element to make this.

<picture>
<source media="(max-width:500px)"
srcset="images/meal-cropped.jpg" />
<source media="(min-width:501px)"
srcset="images/meal.jpg" />
<img src="images/meal.jpg" alt="" />
</picture>

If 500px or less - Serve meal-cropped.jpg
If 501px or More - Serve Meal.jpg
Otherwise the image is meal.jpg
