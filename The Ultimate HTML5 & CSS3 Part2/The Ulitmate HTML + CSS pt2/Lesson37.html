Check Boxes

simply <input type="checkbox"
But checkboxes need a label AFTER, not BEFORE.
input.id === label.for
Can click on Checkbox + Label to acheive result.


But as a best practice, wrap each Input + Label pair in a div of their own.

If we want a checkbox checked by default, simply include the "checked" attribute (Boolean Value)
on the input.
This also works for the "disabled" attribute (Also a Boolean Value)
