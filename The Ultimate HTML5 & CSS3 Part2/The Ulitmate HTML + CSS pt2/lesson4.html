Measurement Units


Absolute Units

Units that are independent - their size doesnt change
Px, ( Inch, Centimetre, Millimetre )
1pixel is 1 pixel, regardless of the size of the screen etc.

Relative Units

Units that are dependant on the size of something else.
%, vh, vw, em, rem.
width: 50% = X% of the size of the parent element
height: 80vh = X% of the size of the Viewport Height
Width: 50vw = x% of the size of the viewport width
width: 10em = x * the font size of the parent element
width: 10rem = x * the font size of the root~(html) element

by default the font size of the HTML element is 16px, but thats a pain to work with

html {
font-size:62.5%;
}

This will take 16px * 62.5% = a new font size of 10px. Easy to work with :)

% by px
vw by vh
rem by em
