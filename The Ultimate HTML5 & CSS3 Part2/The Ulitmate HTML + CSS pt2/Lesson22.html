Background Images

Used to display an image as a background of a webpage.

Background images are declaredin the CSS of the webpage. Using the 5 properties below;

background: url(FilePath);

Sets the background to a given image

background-repeat: repeat-x; | repeat-y; | no-repeat;

Sets the background to repeat on the Horizontal axis, Vertical Axis, or not at all

background-position: 100px 100px; | 100% 50px; | -50px -50px;

Sets the background position 1st Value = Horizontal 2nd Value = Vertical.
2nd Value won't accept a percentage.
---1st Value = Left ---2nd Value = Up

background-size: 100px 200px; | 100% 200px; | cover;

Sets the background size 1st Value = Width 2nd Value = Height

background-attatchment: fixed; |

Sets the background attatchment - Fixed; to be fixed relative to the viewport.
Set bg to fixed / viewheight to 300 = scrollable web page
