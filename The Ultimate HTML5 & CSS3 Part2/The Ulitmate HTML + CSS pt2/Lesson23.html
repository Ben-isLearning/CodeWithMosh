CSS Sprites (Optimization Technique)

Used for icons and logos, instead of sending 5 HTTP requests for 5 small PNG images
We can use a CSS Sprite to combine 5 images into one.
Effectively sending 1 HTTP request for 5 Images.

1) Upload images to (If link doesn't work Google an alternative)
2) Detail Sprite - Generate Sprite - Then Download .zip package
3) Add Sprite image to Image Folder
4) Add CSS to CSS Folder
5) Amend background URL path
6) Link HTML to CSS
7) Insert icons by calling Classes
Only use for logos / icons / Small images
