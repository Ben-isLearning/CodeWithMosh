Supporting High-Density Screens

Physical Resolution: The Size of the physical Screen
Logical Resolution: The Size of screen CSS Uses
Device Pixel Ratio (DPR): Physical Size / Logical Size (2 / 2.1 / 2.2 / 3 )
High desity screen = DPR > 1

Naming Convention for files
Image.jpg Needs to be 400px / 400px /
Image@2x.jpg Needs to be 800px x 800px /
Image@3x.jpg Needs to be 1200px x 1200px / Compression = 7

Start with large image and work your way down!

To serve different images for different screen sizes
Use the attribute srcset="image.jpg 1x, image@2x.jpg 2x, image@3x.jpg 3x,"

This is used for fixed sized images ONLY
that need to be crystal clear - Dont use for every image
