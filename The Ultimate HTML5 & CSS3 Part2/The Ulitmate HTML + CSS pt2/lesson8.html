Grid

Parent Properties

Display:

grid; Generates a grid
inline-grid; generates an inline grid

grid template

grid-template-rows: repeat(x, size); (x = amount of rows, Size in px / % / fn )
grid-template-columns: repeat(x, size); (see above)
grid-template-areas: "name name name" "name . name"
(Name the specific grid area / . = empty grid cell / none = no grid areas defined)

Gap (Effects Items in grid) (Parent property)

column-gap: 20px; (sets gap between col)
row-gap: 10px: (sets gap between rows) gap: row col; (Shorthand - set gap for row then col)

justify-items: (Effects Items in grid) (Parent property)

Justify grid items along the row Axis
justify-items: start; (Items on left)
justify-items: end; (items on right)
justify-items: center; (items in center)
justify-items: stretch; (items all over)

align-items: (Effects Items in grid) (Parent property)

Aligns grid items along the column Axis
align-items: start; (items on top)
align-items: end; (items on bottom)
align-items: center; (items in center)
align-items: stretch; (items all over)
align-items: baseline; (items as they are)

align + justify = place-items; (Effects Items in grid) (Parent property)

place-items: (align value) (justify value);

justify-content: (Effects actual Grid) (Parent property)

This property aligns the grid along the inline (row) Axis
justify-content:start; aligns left
justify-content:end; aligns right
justify-content:center; aligns in center
justify-content:stretch; resizes the grid items fill the full width
justify-content:space-around; gives even amount of space between each item (border gets 1/2)
justify-content:space-between; gives space between items only (no periphperal space)
justify-content:space-evenly; gives even amount of space between each item and the border

align-content: (Effects actual Grid) (Parent property)

This property aligns the grid along the block (column) Axis
align-content:start; aligns grid to top
align-content:end; aligns grid to bottom
align-content:center; aligns grid in center
align-content:stretch; resizes grid items to fill full height
align-content:space-around; even amount of space between each grid item, with half-sized spaces on the far ends
align-content:space-between; even amount of space between each grid item, with no space at the far ends
align-content:space-evenly; an even amount of space between each grid item, including the far ends

place-content: (Effects actual Grid) (Parent property)

Place-content: (align value) (justify value);

Child Properties (Finally!)

Grid-row: X / Y; (starting location / Ending location)
Grid-column: X / Y; (Starting location / Ending location)
grid-area: nameofarea; (No Quotes!)

Example Code

.container {
display:grid;
grid-template: repeat(3,100px) / repeat(2,100px);
grid-template-areas:
"header header"
"sidebar main"
"footer footer";
gap: 10px
border: 3px solid grey;
justify-items: center;
align-items: center;
justify-content: center;
align-content: center;
height: 100vh;
}
.item {
background: gold;
( grid-column: 1 / span 2; Location by number or name, not both)
grid-area: header;
}
