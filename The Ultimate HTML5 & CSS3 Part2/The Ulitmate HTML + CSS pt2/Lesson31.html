Creating a Basic Form

Form Element <form> / <input> / <label>

Forms are used to capture user input and process it server-side.
As such defining input fields, with the <input> element
And ofcourse, each input needs a label, acheived with the <input> element.

To Attatch lables to inputs, simply define and id="x" within the input,
then add the for="x" attribute to the label.
input.id === label.for

Then forms are finished with a set of buttons
In this case, br one button is type="submit" to submit the data to our fake server.
and the other is type="reset" to clear all data.
