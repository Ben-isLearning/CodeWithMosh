Data Lists

Suggestion Lists in HTML

Create a list of suggestions in HMTL by using the <datalist> element
Like Ul's contain Li's... datalist's contain options.
the input must contain the list="" attribute
Equal to the datalist id="" attribute.

The value="" attribute is important serverside when calculations
But the value="" shows up with the selections and its ugly...
A work-around for this is creating a custom classdata-value=""
Then later, using JS we can read the value of data-value as the same as value=""

In addition to this, we should disable the autocomplete attribute
by declaring autocomplete="off"
to ensure that the feild isnt auto-completed based on the users cache or
previous experiences.

The styling of the options list is set by the browser, which can cause
some variation between users..
the browser will override any external CSS
This can be overcome through external CSS with psuedo classes and
JavaScript funtionality (outside scope of course)
