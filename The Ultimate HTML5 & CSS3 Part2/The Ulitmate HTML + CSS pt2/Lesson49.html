BEM

Block Element Modifier

Think of our webpage as a collection of blocks / components.
These blocks contain other blocks can contain elements, and can be modified.

Naming convention for BEM
class="parent__element"
To seperate a Block from an Element
**Only applies to dependant blocks / information**
class="element--modifier"
to seperate a Block from a modifier
**Only applies to a main Blocks**
