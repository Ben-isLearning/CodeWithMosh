When styling webpages with a font-stack, rather than stling each element individually,
Just style the body, and let inheritance do the work for you.
*Fontstack = font-family: font1, font2, genericfont type;*
browser will take font 1, font2, or generic fonttype depending on what it can handle.


When styling font there are 5 properties we need to worry about;
font-family: Used to select the types of font
Ex: font-family: Arial, Helvetica, sans-serif;

font-weight: Used to select the level of boldness
Ex; font-weight: 400;

font-style: Used to select the style of the font
Ex; font-style: italic;

font-size: Used to select the size of the font
Ex; font-size: 2rem;

color: Used to select the colour of the font
Ex; color: darkgrey;


Tl;Dr

body {
font-family: Arial, Helvetica, sans-serif;
font-weight: 400;
font-style: italic;
font-size: 1rem;
color:black;
}
