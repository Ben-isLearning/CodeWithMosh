Sliders

Sliders can be implemented to represent a numeric value.
Just think of every "interactive" survey we've ever done...

this is acheived by <input type="range">
This type of input requires;
the min="" attribute (to set a minimun)
the max="" attribute (to set a maximum)
the value=""attribute (to set a starting position)

We can't show the value of the slider using only HTML and CSS.
But with JavaScript, we can listen to the changes in the value of the slider,
and display the changed value in a seperate label.
(Outside the scope of this course)

File Inputs

File inputs can be implemented to allow users to upload their own documents to the website.
This is acheived by <input type="file">
the "multiple" attribute allows multiple files to be selected at once
the 'accept="" ' attribute restricts the upload to specified file types.
the accept attribute allows multiple values, seperated with a commas.
accept=".svg, .png, .jpg".
or accept="image/*" for all image formats
or accept="video/*" for all video formats
or accept="audio/*" for all audio formats
