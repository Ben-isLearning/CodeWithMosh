Vertical Spacing

is governed by two properties;
margin:
line-height:

Law of proximity = Objects that are closer are perceived to be related
Ensure that this is obeyed by increasing top margin and decreasing bottom margin

If the value given in line-height: is not specificed
It is used as a multiplier (Rule of thumb - 1.5 is a good benchmark)

h1 {
margin: 3rem 0 1rem;
font-size: 1rem;
line-height:1.5;
}

Horizontal Spacing

is governed by three properties:
letter-spacing:
word-spacing:
width:

letter-spacing:1px; sets the space between letters
Use a px value here.. can use -px values to shrink loose font.

word-spacing:5px; sets the space between words
use a px value here..

width:50ch; sets the width of lines by a character amount (ideal size is 50-70chars)
