Positioning

Position: relative;

Position an element relative to its normal position

Normal
Position: Relative; left:20rem;
Has moved from where it was, to where it is.

Position: absolute;

Position an element relative to the parent element

Normal
Position: Absolute; Has moved from where it was, to where it is (Sucked out of DOM + Placed on another layer).
Thats why the Position:fixed; is squished up

Position: Fixed;

Position an element relative to the viewport

Normal
Position:Fixed; Has moved from where it was, to where it is.
