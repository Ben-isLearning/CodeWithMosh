Sizing Elements

Elements can be sized by the height: and width: properties

Only block level element respect these properties
inline elements can be taught respect by adding display:inline-block;

There are two types of box-sizing:
content-box; and border-box;
Content-box; Only cares about the height / width of the content, padding and border values are added ontop
Border-box; Cares about it it all.

Border Box (100x100)
Content Box (100x100)

These two are vastly different. To prevent boxes being mismatched style all boxes to be the same.
Set a rule for all elements, all psuedo before and all psuedo after elements too.

*
*::before
*::after {
box-sizing:borderbox
}
