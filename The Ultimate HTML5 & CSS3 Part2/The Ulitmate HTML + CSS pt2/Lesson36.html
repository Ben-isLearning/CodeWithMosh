Drop-down Lists

Drop-down lists are created with the <select> element
Each value within the list is created with the <option> element
Each option needs a value="x" attribute to be handled server side.


If we want to show a specific option by default, apply the "selected" attribute to said element. (Boolean Value)
(Rather than showing Select a Course... it will show Javascript (4th option))

If we want to enable multiple selections, apply the "multiple" attribute to the select element. (Boolean Value)


If we want to categorize our many options, we can the <optgroup> element
Put the desired options into the optgroup element. Each group needs a label.
