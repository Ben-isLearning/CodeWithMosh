Variables / Custom Properties

Instead of repeating the same line of code, over and over again.
create a custom class
Custom classes are created in the root, by :root{}
custom classes begin with -- but are named however you see fit.

:root {
--colorPrimary: red;
--border-size: 2px;
--border-radius: 10px;
}

The custom properties will be set at the same level as HTML,
and will be inherited by EVERYTHING on the DOM.

These customer properties need to be called by the var();

.one {
background: var(--colorPrimary);
}
