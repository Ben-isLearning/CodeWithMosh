Formatting Text

text-align: is used to control the alignment of text
left; right; center;

text-indent: is used to add indentation on the first line of text
1rem; 2rem; etc

text-decoration: is used to add fancy bits to text
underline; line-through;

text-transform: is used to transform text entirely
lowercase; uppercase; capitalize;

column-count: is used to set number of columns text should be presented in
1; 2; 3; etc

column-gap: is used to set gap between columns of text
2rem; 4rem; etc

column-rule: is used to create a ruled line between columns of text
width style colour; 4px dotted grey;

direction: is used to set the direction text
ltr; rtl;
