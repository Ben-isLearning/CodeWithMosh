Color;

There are 4 (or 6) different ways to use colors.

1) Named colors, like color:blue;
2) RGB, Red, Green, Blue like color:rgb(230,205,15); (0-255)
3) RGBA, As above, but with alpha (opacity) like, color:rgba(230,205,15,0.7); (0-1)
4) HEX, a hexidecimal number, like color:#e6cd10;
5) HSL, Hue, Saturation, Light like color:hsl(53, 87%, 48%);
6) HSLA as above, but with alpha (opacity) like, color:hsla(53,87%,48%,0.3) (0-1)
