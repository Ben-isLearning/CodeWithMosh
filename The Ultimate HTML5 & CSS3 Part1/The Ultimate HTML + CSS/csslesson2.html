Basic Selectors

Syntax: Selector {property:value;}
Basic selectors are ID / Class / Type
<h1 id="xxx"> Hello </h1>
<h2 class="aaa"> Hello </h2>
<h3 > Hello </h3>

These adhere to specificity,
if 2 rules conflict, the most "valuable" rule will win.
100) To select by ID; #xxx {color: green;}
010) to select by Class; .aaa {color: red;}
001) to select by type h3 {color: purple;}

Attribute Selectors
Selector[attribute] | Selects all elements with a specific attribute | a[target]
selector[attribute=value] all elements with attribute with a specific value | a[target=_blank]
selector[attribute~=value] | all elements with the attribute containing "some" | article[title~=some]
Selector[attribute^=value] | elements who's attribute begins with value | a[href="https"]
selector[attribute$=value] | elements who's attribute ends with value | a[href=".pdf"]
selector[attribute*=value] | elements who's attribute contains value | a[href="google"]
