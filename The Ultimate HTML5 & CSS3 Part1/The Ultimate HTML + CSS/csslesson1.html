Provding CSS

CSS Can be provided in three different ways;
Tl;dr... Specificity = Inline > Internal > External

1) Provided externally in style.css with the appropreate
<link> rel="stylesheet" href="style.css" within the Head of HTML

2) Provided internally in the head of the HTML document
<style> p { border: 5px solid black; } </style>
Styles declared internally will be used over those declared externally.

3) Provided inline within the element its self.
<p style="color:blue;"> These are blue words </p>
Styles declared in line will be used over all others.
