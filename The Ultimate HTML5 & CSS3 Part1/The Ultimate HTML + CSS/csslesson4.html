Psuedo-Class selectors
Used to style elements in a particular state

Syntax; selector:psueudo-class{ property:value}

:first-child | Looks for first child of element (Fragile)
:last-child | Looks for last child of element (Fragile)
:first-of-type | Looks for first-of-type p within article
:last-of-type | Looks for last of type
:nth-child(value) | Looks for (value) of the child (odd/even/ 3n / etc)
:visited | Looks for visited elements (a:visited)
:link | Looks for all links
:hover | class is applied when user hovers over element
:focus | class is applied when user tabs to element (use with Hover)
