Containers;

Are essentially places to put things.
Non-semantic containers are; div and span
Lets avoid using them, since they bring no real meaning

Semantic containers are
(Block Level) main, article, figure, header, nav, section, aside, footer,
(Inline Level) mark and time

Rules of thumb;
There should only be a single main
Each section needs a heading
Header represents introductary Content
Aside contains content not directly related to main
Section is used to group related content
Article is used to show a standalone item (like a blogpost)
