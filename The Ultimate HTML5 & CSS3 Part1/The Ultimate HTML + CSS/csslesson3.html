Relational Selectors (Cominator Selectors)

01) Descendant Selector: a space between parent and child
parent child {property:value;}
*Selects all elements that are decendents of the specified element
div p {background-color:yellow;} Will target all decendent Ps within Div

02)Child Selector: a > between the parent and child
parent > child {property:value;}
*Selects all elements that are direct children of the specified element
div > p {background-color:blue;} Will target Ps that are direct chidren of Div

03) Adjacent Sibling Selector: a + between sibling and sibling
Sibling1 + Sibling2 {property:value;}
*Selects all siblings2 that immediately follow the other sibling1
div + p {color: red;} Will target all of the FIRST Ps that immediately follow Divs.

04)General Sibling Selector: a ~ between sibling and sibling
Sibling1 ~ Sibling2 {property:value;}
*Selects all elements that are Later siblings of an element.
div ~ p {color:blue;} Will target all later P siblings of Div.
