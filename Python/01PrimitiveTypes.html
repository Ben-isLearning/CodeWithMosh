Variables

Ex: students_count = 1000
The value: 1000 will be stored in memory
the variable: students_count will be used as a 'reference' for the value.

len() Returns the length of a string

menu = "Sausage Bacon Eggs Beans Spam Toast"
print(len(menu))

Returns 35

print(menu[0])
Returns index [0] of the string
Returns "S"

print(menu[-1])
Returns Index [-1] if the string (1 before zero / Last character)
Returns "t"

print(menu[0:7])
Returns characters from index [0:x] to index [x:7]
Returns "Sausage"

print(menu[0:])
Returns characters from index [0:x] to .... (Unspecified / End)
Returns "Sausage Bacon Eggs Beans Spam Toast"

print(menu[:7])
Returns characters from .... (unspecified / Beginning) to index [x:7]
Returns "Sausage"

print(menu[:])
Returns characters from ....(unsepcifed / Start) to ... (Unspecified / End)
Returns "Sausage Bacon Eggs Beans Spam Toast" (Entire String)

Escape Sequences

"Escape (The next) Character" = \
\" | \' | \\ Comment = #
Linebreak = \n
pie = "Ben's "amazing" pie" ~ Will cause a huge error

Seperate inner and outer quotation.

pie = "Ben's 'amazing' pie" ~ Will be OK!

Or overcome the rules with an "Escape Character" (backslash).

pie = "Ben's \"amazing\" pie" ~ Will be OK!
